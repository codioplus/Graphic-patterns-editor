Graphic Editor

A simple web app called “Graphic Editor” which draws geometric shapes such as circle, square, rectangle, triangle & rhombus. Each shape might have various attributes like border color and size, fill color... etc.

Example: { "shapes": [ { "type": "rhombus", "rhombusX": 15, "border": { "color": "purple", "width":17 } }, { "type": "triangle", "triangleX": 15, "border": { "color": "green", "width":27 } },{ "type": "rectangle", "rectangleWidth": 15, "rectangleLength":40, "border": { "color": "yellow", "width": 7 } },{ "type": "circle", "perimeter": 10, "border": { "color": "red", "width": 7 } },{"type": "square","sideLength": 10, "border": { "color": "#776cff", "width": 19 } }]}

* These fields are required.
